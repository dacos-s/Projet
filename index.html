  • Bitbucket :

    • Vous permet de créer gratuitement et en illimité des dépôts publics mais aussi privés !

    Seule contrainte des dépôts privés, ils sont limités à 5 contributeurs.

  • Gitlab :

    • Vous permet de créer en ilimité des dépôts privé sans nombre limité de contributeurs.

    Avec une protection de branches qui empêche quiconque de forcé ou supprimer la branche sauf les utilisateur ayant l'autorisation.

    Par contre il nécessite de posséder un serveur.

  • CodePlex :

    • Tous les projets sont privé pour une durée maximum de 30 jours, après quoi ils seront supprimés si ils ne sont pas rendu public avant la date d'expiration.

Why Github ?

GitHub est plus grand, et le plus populaire site d'hébergement de code du monde.

Les utilisateurs ont le choix d'utiliser soit Git ou Subversion comme leur VCS (Version Control System), pour gérer, maintenir et déployer des projets de logiciels.

Pour maintenir repos privées, un abonnement mensuel est nécessaire. GitHub vous permet de créer «organisations» qui fonctionnent essentiellement comme une comptabilité régulière, sauf qu'ils ont toujours au moins un compte utilisateur répertorié comme leur propriétaire.
